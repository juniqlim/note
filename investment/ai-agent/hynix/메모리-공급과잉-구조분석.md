# 메모리 공급 과잉 구조 분석 — 전공정/후공정/시클리컬리티

> 2026년 2월 28일 대화 정리

## 핵심 질문과 결론

**Q: 2027~28년 공급이 2026년 대비 얼마나 늘어나는가?**

| 요인 | 2년 누적 증가 |
|------|-------------|
| 신규 팹 (웨이퍼 캐파) | +30~50% |
| 공정 미세화 (1a→1c→1d, Bit/wafer) | +20~30% |
| **Bit 기준 합산** | **+60~80%** |

과거 공급 과잉이 터졌던 시기(2018~19, 2022~23)보다 큰 증가폭.

---

## HBM 적층 증가 = 웨이퍼 더 많이 소모

HBM은 각 층이 별도 웨이퍼에서 제조된 다이를 적층하는 구조. "집적"이 아니라 "쌓기".

| GPU | HBM 세대 | 적층 | DRAM 다이 수/GPU |
|-----|---------|------|-----------------|
| H100 | HBM3 | 8단 | 40개 |
| B200 | HBM3E | 8단 | 64개 |
| Rubin (추정) | HBM4 | 12단 | 96개+ |

8단→12단 = 웨이퍼 소모 **+50%**. 하지만 공장이 +60~80% 늘어나면 이걸 커버하고 남음. HBM 적층 증가만으로는 공급 과잉을 막기 어려움.

---

## 진짜 병목: 후공정 (패키징)

### 웨이퍼는 모자르지 않다

> "웨이퍼를 늘려도 HBM 생산은 안 늘어남. TSV·적층·MR-MUF 패키징이 병목. 웨이퍼를 늘리면 HBM은 그대로인데 범용만 넘침." — hynix.md

### 후공정은 메모리 회사가 직접 수행

```
[1단계] HBM 스택 (SK하이닉스/삼성/마이크론)
  DRAM 다이에 TSV → 다이 적층 → MR-MUF/TC-NCF 봉지

[2단계] GPU+HBM 합체 (TSMC)
  GPU 다이 + HBM 스택을 CoWoS 패키징
```

외주가 아닌 자체 수행. 전용 시설 신축 필요, 리드타임 1.5~2년.

### 전공정 vs 후공정 캐파 격차 (2026년)

| 회사 | 전공정(웨이퍼) | 후공정(패키징) |
|------|-------------|-------------|
| SK하이닉스 | **+100%** (월 30만→60만) | **+13%** (월 15만→17만) |
| 삼성 | **+47%** (월 17만→25만) | P5 실가동 2027말 |
| 마이크론 | PSMC 팹 인수 | 싱가포르 2027부터 본격 |

전공정은 +50~100% 늘리는데, 후공정은 +13~20%밖에 못 늘림.

### 결과

- **HBM**: 후공정 병목으로 타이트 유지 → 가격 방어 가능
- **범용 DRAM (DDR5 등)**: 늘어난 웨이퍼가 범용으로 흘러감 → 공급 과잉 → 가격 하락

## 하이닉스는 TSMC처럼 비시클리컬이 될 수 있는가?

### TSMC가 비시클리컬이 된 진짜 이유

TSMC도 2000년대까지는 시클리컬이었음. UMC, 차터드, GF 등과 경쟁.

전환점:
1. 2014년: 애플 독점 위탁 → R&D 자금 몰림
2. 2018년: GF 7nm 포기 → 첨단 경쟁자 축소
3. 2022~: 삼성 3nm 수율 부진 → 사실상 독점

**"주문 생산이라서"가 아니라, "경쟁자가 기술적으로 탈락해서" 비시클리컬이 된 것.**

### HBM의 TSMC 유사성

| | 범용 DRAM | HBM |
|---|---|---|
| 표준 | JEDEC → 3사 호환 | 고객별 커스텀 스펙 |
| 고객 전환 | 쉬움 | 어려움 (퀄 테스트 6개월+) |
| 가격 결정 | 수급 → 시클리컬 | 장기 계약 → 안정적 |
| 경쟁 구도 | 3사 대체 가능 | 하이닉스 62%, 대안 부족 |

### 결론: "약한 시클리컬"이 현실적

**비시클리컬 불가 이유:**
- 삼성이 HBM4 퀄 통과 → **경쟁자가 탈락하지 않음** (TSMC와 결정적 차이)
- 범용 DRAM 45%를 버릴 수 없음 → 시클리컬 부분 잔존
- 후공정 병목도 2027~28년 3사 증설 완료 시 해소 → 다시 공급 경쟁

**과거보다 진폭 축소 근거:**
- HBM 이익 쿠션 (이익의 55%+)
- 후공정 병목이 진입장벽 역할
- 장기 계약 + 완판 구조

> "사이클은 옴. 하지만 HBM 이익 쿠션 + 수요 탄력성 조정으로 진폭이 작아질 수 있음. 시장이 과거 진폭(-70%)을 가정하고 fPER 5~6배를 준다면, 실제 진폭이 -35~40%에 그칠 때 리레이팅 기회." — hynix.md
