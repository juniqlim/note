# 실리콘투 vs ABW(에스아시아) B2B 사업 비교

## 1. B2B의 출발점이 정반대

| | 실리콘투 | ABW (YesAsia) |
|---|---------|---------------|
| 태생 | **B2B가 본업** (CA 매출 88~93%) | **B2C(YesStyle)에서 파생**된 B2B (19→32%) |
| B2C 비중 | PA 매출 3~4%로 부수적 | B2C가 여전히 68% |

실리콘투는 처음부터 도매 플랫폼(Stylekorean Wholesale)이 핵심이고, ABW는 YesStyle이라는 B2C 인프라 위에서 B2B를 키워가는 구조.

## 2. 고객 구성이 근본적으로 다름

**실리콘투**: 180개국+ 중소 바이어 다수, "다품종 소량 판매"
- CA 상위 3개사 매출 비중 = 18.2% (2024), 나머지 69.6%가 "기타"
- 수많은 중소 리테일 파트너에게 온라인 Wholesale 플랫폼으로 공급

**ABW**: 대형 리테일 체인 집중
- Ulta, TJX, Burlington (미국), Primark, Superdrug (영국), OVS (이탈리아), 7-Eleven (태국) 등
- 21개 리테일러, 17개국 (1H2025 기준)
- 소수 대형 거래처에 집중 → 1H 급증/2H 둔화 계절성의 원인

## 3. 오프라인 진출 전략이 다름

**실리콘투**: 온라인 Wholesale 플랫폼 중심. 오프라인은 마케팅 행사 수준
- 해외 파트너들이 알아서 현지 리테일에 재판매하는 구조

**ABW**: **직접 대형 오프라인 체인과 계약** (Ulta 매대, Primark 매장 등)
- 빅데이터 분석으로 리테일 파트너의 재고 최적화, 트렌드 예측까지 지원
- Kiokii(캐나다 뷰티체인) 같은 오프라인 전용 파트너십

## 4. 데이터 무기가 다름

**실리콘투**: 180개국 판매 데이터 기반 국가별 제품 큐레이션 + 68개국 3만명 인플루언서

**ABW**: B2C 40만 인플루언서 + 소비자 행동 데이터를 B2B 영업에 활용. "이 제품이 YesStyle에서 잘 팔리니 당신 매장에도 넣으세요"라는 논리.

## 5. 브랜드 서비스 범위

**실리콘투**: 브랜드사에 대한 **엔드투엔드 서비스** 제공
- BM(브랜드관리): 소싱, 수출서류, 통관, 인허가
- CA: 도매 판매
- 풀필먼트: 큐텐/쇼피 등 오픈마켓 입점 대행, 배송대행(Drop Shipping)
- 마케팅: 자체 스튜디오, SNS 콘텐츠 제작

**ABW**: **리테일 체인 중심 공급** + 빅데이터 분석 서비스
- 리테일 파트너에게 시즌별 제품 공급 + 마케팅 플랜 데이터 지원
- 브랜드사 대상 서비스보다는 바이어(리테일) 대상 서비스에 집중

## 요약: 핵심 차이

| | 실리콘투 | ABW |
|---|---------|-----|
| B2B 모델 | 온라인 도매 플랫폼 → 다수 중소 바이어 | 대형 리테일 체인 직접 납품 |
| 판매 채널 | 온라인(Stylekorean Wholesale) 중심 | 온라인 + **오프라인 체인** |
| 서비스 대상 | 브랜드사 + 해외 바이어 양쪽 | 주로 해외 리테일 바이어 |
| 고객 집중도 | 분산 (롱테일) | 집중 (소수 대형) |
| 데이터 원천 | B2B 판매 데이터 | B2C 소비자 데이터 |

둘 다 K-뷰티 유통이지만, 실리콘투는 "수많은 중소 파트너에게 온라인 도매 플랫폼으로 공급"하는 모델이고, ABW는 "B2C 데이터를 무기로 대형 오프라인 체인을 직접 공략"하는 모델. 같은 B2B지만 고객층과 접근법이 상당히 다르다.
