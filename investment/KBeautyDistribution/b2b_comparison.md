# 실리콘투 vs ABW(에스아시아) B2B 사업 비교

## 1. B2B 매출 비중

| | 실리콘투 | ABW (YesAsia) |
|---|---------|---------------|
| B2B 비중 | CA 매출 88~93% | 19→32% (1H2023→1H2025) |
| B2C 비중 | PA 매출 3~4% | B2C(YesStyle) 68% (1H2025) |

## 2. 고객 구성

**실리콘투**: 180개국+ 중소 바이어 다수
- CA 상위 3개사 매출 비중 = 18.2% (2024), 나머지 69.6%가 "기타"

**ABW**: 대형 리테일 체인 중심
- Ulta, TJX, Burlington (미국), Primark, Superdrug (영국), OVS (이탈리아), 7-Eleven (태국) 등
- 21개 리테일러, 17개국 (1H2025 기준)

## 3. 오프라인 채널

**실리콘투**:
- 모이다(MOIDA) 직영 편집숍 운영 (2025년 말 10~20개, 2026년 50개 목표)
- 영국 Boots 2,500개+ 매장 입점
- 미미박스 파트너십으로 미국 대형 유통 14개 채널, 4,000개+ 매장 진입 경로 확보

**ABW**:
- 직접 대형 오프라인 체인과 계약 (Ulta 매대, Primark 매장 등)
- Kiokii(캐나다 뷰티체인) 등 오프라인 전용 파트너십

## 4. 데이터 자산

**실리콘투**: 180개국 판매 데이터, 68개국 3만명 인플루언서 네트워크

**ABW**: B2C 40만 인플루언서, 소비자 행동 데이터

## 5. 브랜드 서비스 범위

**실리콘투**: 브랜드사에 대한 엔드투엔드 서비스
- BM(브랜드관리): 소싱, 수출서류, 통관, 인허가
- CA: 도매 판매
- 풀필먼트: 큐텐/쇼피 등 오픈마켓 입점 대행, 배송대행(Drop Shipping)
- 마케팅: 자체 스튜디오, SNS 콘텐츠 제작

**ABW**: 리테일 체인 중심 공급 + 빅데이터 분석 서비스
- 리테일 파트너에게 시즌별 제품 공급 + 마케팅 플랜 데이터 지원

## 요약

| | 실리콘투 | ABW |
|---|---------|-----|
| B2B 모델 | 온라인 도매 플랫폼 + 오프라인 확장 | 대형 리테일 체인 직접 납품 |
| 판매 채널 | 온라인(Stylekorean) + 오프라인(모이다, Boots, 미미박스) | 온라인 + 오프라인 체인 |
| 서비스 대상 | 브랜드사 + 해외 바이어 양쪽 | 해외 리테일 바이어 |
| 고객 집중도 | 분산 (상위 3사 18.2%) | 집중 (소수 대형) |
| 데이터 원천 | B2B 판매 데이터 | B2C 소비자 데이터 |

같은 B2B지만 고객층과 접근법이 상당히 다르다.

> 출처: 실리콘투 DART 사업보고서(2024 연간, 2025 반기), YesAsia Holdings 실적 발표(2024 연간, 2025 중간, 2025 Profit Alert)
