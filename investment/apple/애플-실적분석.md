## 6. 밸류에이션

| 지표 | 수치 |
|------|------|
| 시가총액 (2026.02) | ~$3.76T |
| Trailing PER | ~32.3x |
| Forward PER | ~30.4x |
| 5년 PER 추이 | 22.5x(2022) → 28.2x(2023) → 38.3x(2024) → 34.0x(2025) |
| PSR | 9.1x |
| FCF Yield | 2.6% |
| ROA | 30.9% |
| ROIC | 70.7% |

### PER이 높은 이유

1. **Services 비중 확대** (26% → 고마진, 반복매출): SaaS형 밸류에이션 프리미엄 부여
2. **자사주 매입 효과**: 매년 주식수 ~2.5% 감소 → EPS 성장 > 순이익 성장
3. **CapEx 경량 모델**: $12.7B CapEx로 $133B 영업이익 → FCF 전환율 극도로 높음
4. **AI 시대 하드웨어 해자**: 25억대 설치 기반, 생태계 락인
